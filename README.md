# صفحة مسار هندسة وعلوم حاسب

صفحة تعريفية بمسار "هندسة وعلوم حاسب" (ثانوي)، بتشرح المسار وتدعو الطلبة للانضمام لجروب واتساب لمشاركة الملخصات والمذاكرة الجماعية.

## الملف

- `index.html` — الصفحة كاملة (HTML + CSS + JS) في ملف واحد، مفيش أي تبعيات خارجية غير خطوط Google Fonts.

## المحتوى

| القسم | الوصف |
|---|---|
| **Hero** | عنوان رئيسي + خلفية متحركة (شبكة نقط متصلة بخطوط، شكل دوائر/شبكة) |
| **عن المسار** | شرح مختصر لطبيعة المسار (هندسة + علوم حاسب) وأقسامه |
| **شروط القبول** | المواد المطلوبة: تانية ثانوي (برمجة، انجليزي، تاريخ، عربي) وتالتة ثانوي (رياضة مستوى رفيع، فيزياء مستوى رفيع) |
| **الجروب** | فايدة الجروب (ملخصات، نقاش، تحديثات) + كارت قوانين واضح (ممنوع تجاوزات، احترام، من غير سبام) |
| **CTA** | دعوة نهائية للانضمام لجروب الواتساب |

## رابط جروب الواتساب

الرابط متربط في مكانين:
- زرار "انضم للجروب" في الـ navbar (فوق)
- زرار "دخول جروب الواتساب" في قسم الـ CTA (في آخر الصفحة)

كلاهما يفتحوا في تاب جديد (`target="_blank"`).

لو الرابط اتغير، ابحث عن `chat.whatsapp.com` في `index.html` وغيّره في الموضعين.

## التصميم

- **الخطوط**: Cairo (العناوين) + Tajawal (النصوص) + JetBrains Mono (تفاصيل تقنية صغيرة)
- **الألوان**: خلفية كحلي غامق (`#070914`)، مع لمسات سيان (`#3df2e0`) وبنفسجي (`#8b5cf6`) وروز (`#ff5fa2`)
- **الاتجاه**: RTL بالكامل (`dir="rtl"`)
- **الحركة**: خلفية كانفاس متحركة (نقط متصلة) + ظهور تدريجي للعناصر عند السكرول، مع احترام `prefers-reduced-motion`

## التعديل

كل النصوص والألوان والمواد موجودة مباشرة في `index.html`. للتعديل على:
- **المواد الدراسية** → قسم `<section>` اللي فيه `class="network-wrap"`
- **قوانين الجروب** → `class="rules-card"`
- **الألوان** → متغيرات CSS في أول الملف (`:root { --cyan, --violet, --pink, ... }`)

## الفتح

افتح `index.html` في أي متصفح مباشرة، أو ارفعه على أي استضافة ستاتيك (GitHub Pages, Netlify, Vercel) من غير أي تعديل.
